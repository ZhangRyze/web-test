## 调用说明
全局可调用，通过``` <app-query-input></app-query-input> ```调用

## prop说明

#### *property* **必填**

查询条件的key

#### *placeholder* **选填**

用于显示无选择时输入框内容，如果需要通过``` placeholder="xxx" ```调用。

#### *operator* **选填**

查询条件key与value的关系，默认是等于也就是"like"

#### *paramType* **选填**

查询条件的数据类型，默认为Object,暂无其他

#### *label* **选填**

查询条件的名称
